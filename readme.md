## 🎯 О проекте

**Gate Alpha Agent** — автономный торговый бот, который:

✅ **Сканирует** секцию Alpha Gate.io на предмет высокорискованных активов с потенциалом  
✅ **Анализирует** сигналы через локальную LLM (Ollama) без отправки данных вовне  
✅ **Определяет** точку входа, стоп-лосс и размер позиции с учётом риск-менеджмента  
✅ **Генерирует** сетку ордеров на выход (grid take-profit) для фиксации прибыли  
✅ **Работает** в режиме testnet для безопасного тестирования стратегий  

> ⚠️ **Важно**: Активы в секции Alpha — высокорискованные мем-токены. Используйте только средства, которые готовы потерять.
